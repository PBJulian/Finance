# Finance
Repositorio académico de Introducción a las Finanzas y la Empresa. Contiene PDFs, apuntes, guías, ejercicios y material del semestre sobre intereses, anualidades, bonos, valuación, riesgo, rendimiento, portafolios, CAPM y más. Su objetivo es centralizar el contenido del curso y mantener consistencia con la metodología y fórmulas vistas en clase.
